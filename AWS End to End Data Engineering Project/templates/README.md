**Project Description**

This project is a student data entry form. It allows schools or teachers to record important details such as a **student’s registration number, name, class, and marks in subjects like Math, English, Science, and Computer.** 

The form is designed to be easy to use, with a clean layout and a single “Submit” button to save the information. It works well on different devices, making it a straightforward way to collect and manage student records online.
